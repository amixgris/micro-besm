Версия микрокода 1992 года, с изменениями при наладке заводского экземпляра.

(1) Размер микрокоманды расширен с 112 до 120 бит.
    Добавлены два новых поля:
        CSMPY - 1 бит - сигнал выборки умножителя;
        MPYF - 3 бита - функция умножителя.
    Эти сигналы нигде не используются.

(2) Добавлены адреса PROM для команды RETS - вероятно,
    ошибочно пропущенные раньше.

(3) При обработке прерываний поле CYSTR используется для изменения
    длительности тактового импульса. В прерыдущей версии микрокода
    тактовый импульс всегда имел одинаковую длительность.
